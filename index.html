The de'Barbari Map

This map of Venice was created by Jacopo de'Barbari in the year 1500. For more details, refer to the map's Venipedia entry. This tool allows you the explore the Venice Project Center's extremely high quality scan.
You can enable and disable different color overlays for different landmarks on the site. When this is done, the search will be modified so that only landmarks highlighted in the selected overlays will appear in the autocompletion dropdown.
When logged in, additional features can be added. Once logged in, a pencil icon will appear in the toolbar. Clicking this will allow you to draw a new polygon. Further instructions are given once the drawing process is started. Once a polygon is complete, metadeta such as the feature's name and must be entered. Then the feature can be saved for future viewing and searching.
Editing shapes is also possible. Simply select a feature (a popup will appear when selected), and click the pencil icon. This will put points around the polygon which can be edited. New points can be added and old ones can be removed. When satisfied with the placement of the points, click the pencil icon again. Now, the same dialog from creating new features will appear, which you can use to edit its name, type, and other data. If you want to remove the feature, click "Delete". WARNING: This will permanently delete the feature! If you simply want to delete a feature without editing its points, simply select the feature, click the pencil icon twice, and click "Delete".
